# Introduction

This Lab adds [Atlas Device Sync](https://www.mongodb.com/atlas/app-services/device-sync) to an already existing iOS app that stores data in a local database, so we can store that data in MongoDB Atlas.

|Lab goals|Learn how to add Atlas Device Sync to an existing SwiftUI iOS app that uses the Realm SDK to store data locally|
|-|-|
|What you'll learn|How to create a free MongoDB Atlas Cluster |
|                 |How to create an App Service and cofigure it with Atlas Device Sync|
|                 |How to change your existing iOS code to sync data with MongoDB Atlas|
|Prerequisites| See [Prerequisites](category/prerequisites)|
|Time to complete|3 hours|
